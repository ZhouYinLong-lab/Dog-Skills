# MarkItDown — 微软开源文档转换神器

> 16 万 Star · MIT 开源 · PDF/Word/PPT 一键转 Markdown

## 这是什么？

微软 AutoGen 团队开源的 Python 工具，把 20+ 种文件格式转成干净 Markdown——标题层级、列表、表格、链接完整保留。人看得舒服，AI 读得顺畅。

## 安装

```bash
pip install 'markitdown[all]'
```

## 一行转换

```bash
markitdown 项目报告.pdf -o 输出.md
```

## 谁适合用

- 🗄️ **Obsidian 用户**：外部文档一键入库
- 🤖 **RAG 开发者**：文档预处理省 80% 清洗工作
- 💼 **职场人**：统一团队文档格式
- 🎓 **学生/研究者**：课件转笔记、文献整理

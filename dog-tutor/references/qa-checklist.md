# 质量验收清单 (QA Checklist)

> Dog-Tutor 交付物必须通过此清单。阻断性缺陷出现任一即判定不通过。

---

## 阻断性缺陷 (一票否决)

- [ ] **B1**: 列表符号粘连正文(如 `...标签。- 列表项` — 最高级别格式错误)
- [ ] **B2**: `**粗体**` 标记前后缺少空格(正则 `[^\s]\*\*[^\s\*]` 可检测)
- [ ] **B3**: 代码块/Admonition(`!!!`) 前后缺少空行
- [ ] **B4**: 章节内容与大纲结构不符
- [ ] **B5**: 教程未包含验证步骤(每章至少1个)

---

## 维度 1: 格式规范 (权重 15%, 满分 15)

| # | 检查项 | 通过标准 | 得分 |
|---|--------|----------|------|
| 1.1 | MkDocs 兼容性 | `mkdocs build --clean` 无报错 | /3 |
| 1.2 | 空行隔离 | 所有列表/代码块/Admonition 前后有空行 | /3 |
| 1.3 | 粗体空格隔离 | 所有 `**text**` 前后各有一个空格 | /3 |
| 1.4 | 标题层级 | 一级→二级→三级,不跳级 | /3 |
| 1.5 | 视觉层次 | Admonition/代码块/Tabbed Content 使用恰当 | /3 |
| | | **小计** | **/15** |

# 最终验收清单 (QA Checklist)

> 所有 Dog-Frontier 交付物必须通过此清单的全部检查项。
> 阻断性缺陷出现任一即判定不通过,不计算总分。

---

## 阻断性缺陷 (一票否决)

- [ ] **B1**: 移动端(375px)有水平滚动条
- [ ] **B2**: 任何文字对比度 < 3:1(包括 disabled 状态)
- [ ] **B3**: 表单输入框无可见 label (placeholder-only 视为不合格)
- [ ] **B4**: 交互元素(按钮/链接/表单)无键盘可达性
- [ ] **B5**: 输出包含未标注来源的上游技能内容(抄袭嫌疑)

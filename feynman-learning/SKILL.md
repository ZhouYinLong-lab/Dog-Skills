# 费曼学习法 — Feynman Learning Technique

把费曼四步学习法装进 AI，20 分钟完成深度理解和终身记忆。

## 费曼四步法

```
Step 1: 选择概念    →  确定要学习的目标概念
Step 2: 教给别人    →  用最简单的话解释，假装讲给一个完全不懂的人
Step 3: 发现缺口    →  识别解释中卡住或模糊的地方，回到原始材料学习
Step 4: 简化类比    →  用生活化的比喻和故事重新组织，让解释更简单有力
```

## AI 辅助工作流

### Phase 1: 概念输入
接收用户要学习的概念/知识点，确认学习范围。

### Phase 2: 模拟教学（第一轮）
AI 扮演「完全不懂的学生」，让用户尝试解释概念：
- 用户解释 → AI 追问「为什么」「什么意思」「举个例子」
- 记录用户解释中的模糊点、跳跃点和卡顿点

### Phase 3: 缺口诊断
分析用户解释中的薄弱环节：
- 哪些地方用了术语但不会解释？
- 哪些逻辑链条断裂？
- 哪些例子脱离实际？

### Phase 4: 回归原始材料
引导用户回到原始材料，针对缺口补充学习：
- 定位关键段落或公式
- 用多个来源交叉验证
- 重写不清晰的部分

### Phase 5: 简化输出（第二轮教学）
- 用生活场景重新类比
- 一句话总结核心
- 建立一个「对 5 岁小孩也能讲明白」的版本

### Phase 6: 长期记忆固化
- 生成间隔复习时间表
- 建立与已有知识的关联网络
- 生成自测题

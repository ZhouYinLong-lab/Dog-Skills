# Torn Paper Collage Poster — 撕纸拼贴海报

> 手工感极强的编辑式拼贴海报：撕纸层叠 + 粗糙排版 + 邮票 + 胶带 + 贴纸 + 复印机纹理。

## 风格元素

- **撕纸层叠** — 不规则边缘、纤维毛边、重叠阴影
- **粗糙排版** — 大小字号混排、倾斜、压印效果
- **邮票和邮戳** — 圆形锯齿边邮票、日期邮戳
- **胶带和贴纸** — 泛黄透明胶带、彩色贴纸、贴痕
- **复印机纹理** — 黑白复印机噪点、扫描线、对比度增强
- **手写注释** — 潦草的批注、箭头、圈画

## 安装

```bash
npx skills add MagicCube/agentara --skill torn-paper-collage-poster -y -g
```

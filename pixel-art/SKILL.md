# Pixel Art — 像素画工作室

> 内置设计系统的口袋像素画工具，输出真正的网格对齐像素 PNG/SVG。

## 支持的风格

- **复古游戏**：8-bit / 16-bit 经典游戏风格
- **等距视角**：Isometric 像素建筑和场景
- **角色精灵**：Character Sprite，含行走帧动画
- **场景绘制**：像素风景、城市场景
- **物品图标**：像素道具、UI 图标

## 安装

```bash
/plugin marketplace add sorawit-w/agent-skills
/plugin install agent-skills@sorawit-w
```

## 触发示例

- "画一个中世纪港口的像素画，黄昏时分有灯塔"
- "做一个等距视角的像素咖啡店"
- "create a pixel-art character sprite for my RPG game"

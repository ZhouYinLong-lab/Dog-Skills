# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in any Dog-Skills skill, please report it via email to **liuhanzhi514@gmail.com**.

**Do NOT open a public issue.** We will respond within 48 hours and keep you updated throughout the disclosure process.

## Scope

This policy covers all skill directories in this repository. Each skill's security posture depends on its specific tools and integrations. Skills that involve code execution, web fetching, or external API calls should be reviewed carefully before use.

## Third-Party Skills

Many skills in this repository are wrappers or adaptations of third-party projects. Security issues in upstream projects should be reported to their respective maintainers. We track upstream sources in each skill's `SKILL.md` frontmatter under `metadata.source`.

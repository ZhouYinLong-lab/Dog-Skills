# Architecture Diagram Generator

创建精美的深色主题技术架构图，输出为自包含 HTML+SVG 文件。

## 特性

- 🎨 深色主题 + JetBrains Mono 字体
- 🔍 内置导出功能：一键 Copy PNG / Download PNG / Download PDF
- 🏗️ 多轮结构化提问，确保真正理解系统架构
- 🎯 7 种语义配色覆盖 Frontend / Backend / Database / Cloud / Security / Message Bus / External

## 版本

v2.0 — 基于 [Cocoon AI v1.1](https://github.com/Cocoon-AI/architecture-diagram-generator) 深度优化，新增多轮结构化发现协议。

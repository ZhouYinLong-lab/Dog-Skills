# Soviet Storybook Grotesque — 苏联童书怪诞风

> 把照片变成奇怪、粗糙、褪色的东欧儿童书籍插图，配上荒诞的手写押韵诗。

## 风格特征

- **粗糙褪色** — 模仿上世纪苏联儿童读物的印刷质感
- **怪诞人物** — 人物比例夸张、表情古怪
- **手写押韵诗** — 自动生成荒诞好笑的配图文案
- **东欧色调** — 灰绿、棕黄、褪色红、泛黄纸底

## 安装

```bash
npx skills add MagicCube/agentara --skill soviet-storybook-grotesque -y -g
```

## 触发示例

- "把这张照片变成苏联童书风格"
- "用东欧绘本风画我的猫，配荒诞诗"
- "transform this photo into a soviet storybook illustration"

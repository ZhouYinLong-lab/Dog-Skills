## 各部分写作要求

### 0. 本章考试相关性总结

**必须掌握**：列出本章一定会考的内容，每一项必须关联至少一个证据（往年卷题号、课后习题题号、教材例题、或用户要求）。

**已剔除内容**：列出被剔除的内容及原因。原因必须来自阶段 1 的分析结果。

**被压缩保留的内容**：列出压缩保留的内容、压缩方式、以及为什么不能完全剔除（通常是前置知识依赖）。

**本章常见考法**：列出本章在考试中出现的典型考法，每种考法用一两句话描述。列出的每一种考法，必须在"典型题型总结"中至少有一个对应题型。

### 1. 本章知识结构

不列知识点清单，而是用连贯的自然语言说清楚：
- 本章知识之间的逻辑链条
- 先讲什么、后讲什么、为什么是这个顺序
- 不同知识点之间如何互相依赖

目标是让学生读完这段就能建立本章的 mental model。

### 2. 核心知识点

每个知识点必须覆盖以下维度，不能只罗列公式：

- **概念本质**：这个概念到底在说什么
- **为什么引入**：解决什么问题才需要这个概念
- **前后关系**：它和前后知识点有什么联系
- **公式每一项的含义**：物理意义或数学意义
- **公式为什么长这样**：直观理解，不是推导
- **如何理解和记忆**：记忆技巧或类比
- **使用条件**：什么条件下可以用
- **不能用的情况**：什么条件下不能用
- **做题识别信号**：看到什么条件/问法就该用它
- **常见误区**：学生最容易犯的错
- **例题或反例**：至少一个，简洁但能说明问题

### 3. 典型题型总结

题型数量不得少于"常见考法"数量的 80%。

每个题型包括：
- **考查内容**：这个题型考什么知识点/方法
- **做题步骤**：编号的步骤列表，每一步是可执行的动作
- **例题**：来自往年卷或习题，标注来源
- **答案**：完整答案
- **详细思考过程**：第一步为什么这样做 → 中间关键步骤为什么成立 → 答案如何检查 → 常见错误

### 4. 本章复习清单

用 "我能..." 句式，让学生可以逐条自测。覆盖本章所有必须掌握的内容。

## 例题选择规则

例题来源优先级：
1. 往年卷中能代表高频考点的题
2. 课后习题中能覆盖核心方法的题
3. 教材例题中最典型的题
4. 必要时自行构造非常短的小例题

原则：简洁但全面——不能堆砌但必须覆盖主要题型。

## 内容保留规则

- 尽量保留教材原文：定义、定理、公式、符号说明、重要结论、核心解释、典型例题题干、关键推导步骤、图表标题和说明
- 允许重排结构使逻辑更清晰：先概念 → 再公式 → 再用途 → 再例题
- 允许合并教材中分散在不同位置但对同一知识点的讨论
- 允许补充教材某处没讲清楚但例题/习题中体现的思想
- 禁止改变原意

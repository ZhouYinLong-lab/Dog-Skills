# 报告模板

## 完成报告模板

每章通过所有质量门禁后，使用以下模板汇报：

```md
# 第 X 章完成报告

## 已生成文件

- output/tutorials/chXX.md
- output/exercise_classes/chXX.md
- output/audit/chXX_coverage.csv
- output/audit/chXX_md_check.txt
- output/audit/chXX_text_quality_check.txt
- output/audit/chXX_image_audit.csv（如有图片）
- output/assets/chXX/（如有图片）

## 保留内容

- 知识点 A：原因（往年卷高频 / 习题覆盖 / 用户要求）
- 知识点 B：原因
- ...

## 压缩保留内容

- 知识点 C：压缩方式 + 保留原因（前置知识依赖等）
- ...

## 剔除内容

- 内容 D：原因 + 证据（往年卷未出现 / 用户指定剔除）
- ...

## 图片质量检查结果

（如有图片）
- figure_XX_01.png：✅ 通过 / ❌ 需人工复核
- figure_XX_02.png：✅ 通过 / ❌ 需人工复核
...

（如无必要图片）
本章无必要图片，未生成图片文件夹；已在自检文件中说明。

## 文本质量检查结果

- 考法-题型覆盖率：X/Y = Z%
- 所有必须掌握考点均有证据：是 / 否
- 所有核心知识点解释完整：是 / 否
- 所有例题讲解完整：是 / 否

## 习题课覆盖情况

- 覆盖往年题数量：X 道
- 覆盖知识点数量：Y 个
- 高频题型覆盖：全部 / 缺少 Z

## 自检结果

- Markdown 格式：通过 / 有问题
- 图片链接：全部有效 / 有缺失
- KaTeX 公式：无报错 / 有报错

## 需要用户确认的问题

- 问题 1
- 问题 2
- （没有则写"暂无"）
```

## 未完成问题清单模板

如果章节未通过质量门禁，使用以下模板汇报：

```md
# 第 X 章未完成问题清单

## 未通过项目

- 门禁项 1：具体说明
- 门禁项 2：具体说明
- ...

## 需要修复的图片

- 图片 A：问题描述 + 修复方案
- ...

## 需要补充的题型

- 题型 A：对应哪个常见考法
- ...

## 需要补充证据的考点

- 考点 A：缺少什么证据
- ...

## 需要重写或加深解释的知识点

- 知识点 A：当前问题 + 改进方向
- ...

## 下一步修复计划

1. 步骤 1
2. 步骤 2
3. ...
```

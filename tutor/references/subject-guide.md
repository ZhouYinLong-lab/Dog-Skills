# 学科适配指南

本 skill 以数学类课程为默认基准。处理其他学科时，在阶段 2 开始前查阅对应章节的特殊要求。

## 数学类（微积分、线性代数、概率统计等）

默认模式，所有规则适用。特别关注：
- 公式格式严格遵循 `references/formula-rules.md`
- 知识点讲解重点：概念本质 → 公式含义 → 使用条件 → 做题识别
- 图片通常较少，函数图像、几何解释类图片才需截取

## 电路与电子技术（模拟电路、电路分析、电子技术等）

处理此类章节时，每个电路知识点必须额外说明：

### 元件作用
- 每个电阻的作用（偏置、限流、负载、反馈等）
- 每个电容的作用（耦合、旁路、滤波、补偿等）
- 电流源、电压源的作用
- 晶体管、二极管、MOS 管等器件的控制关系

### 等效分析
- 直流分析时哪些元件等效为什么
- 交流小信号分析时哪些元件等效为什么
- 为什么可以建立该等效模型

### 放大器分析
- 输入电阻、输出电阻、电压增益、电流增益如何理解
- 公式每一项的物理意义
- 如何从电路结构判断公式形式

### 器件专题（如 BJT）
- BJT 的本质控制性质
- 电流关系的实际意义
- 为什么可以使用小信号模型
- 小信号模型中每个等效元件代表什么
- 哪些公式考试常用，哪些可以剔除
- 基本解题流程
- 覆盖关键考点的简洁例题

### 器件专题（如 MOSFET）
- 共源极放大电路中各个电阻的作用
- 电流源的放置原因
- 偏置电路的目的
- 小信号等效电路的画法
- 输入/输出电阻、电压增益公式的理解
- 覆盖关键考点的简洁例题

### 电路图处理
电路图是必须保留的核心图像，必须按 `references/image-rules.md` 从 PDF 截取局部清晰图，不得遗漏。

## 大学物理（力学、电磁学、光学、热学等）

处理此类章节时注意：

### 物理图像
- 受力分析图的截取和保留
- 场线图、光路图、波形图的截取和保留
- 物理过程示意图的必要性判断

### 讲解重点
- 物理定律的适用条件和边界
- 守恒律的使用前提
- 公式的物理意义（不只是数学形式）
- 典型模型（质点、刚体、理想气体、点电荷等）的适用条件

### 计算题
- 先分析物理过程，再建立方程
- 明确每一步用了哪个物理定律
- 注意单位制和量纲检查

## 计算机/编程类

处理此类章节时注意：

### 算法
- 时间/空间复杂度分析
- 算法执行流程的逐步跟踪
- 边界条件和特殊输入的处理

### 代码
- 代码执行流程的逐步解释
- 关键变量的状态变化
- 常见 bug 和调试方法

### 图表
- 流程图、状态图、架构图的截取
- 数据结构示意图的必要性判断

## 化学/生物类

处理此类章节时注意：

### 化学
- 反应机理的逐步解释
- 官能团的反应特性
- 反应条件的影响
- 结构式的清晰表示（可能需要截图）

### 生物
- 结构-功能关系的解释
- 过程的逐步描述（如信号通路）
- 实验条件和对照设计
- 通路图、结构图的截取

## 通用原则

无论什么学科，以下原则不变：
1. 从材料出发，不凭记忆生成
2. 以学生看懂为目标，不只罗列知识点
3. 证据驱动的内容分类（保留/压缩/剔除）
4. 完成前必须通过质量门禁

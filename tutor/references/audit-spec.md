# 审计文件格式规范

每章必须生成的审计文件及其格式定义。

## 1. 考点映射文件

**路径**：`output/audit/past_paper_topic_map.csv`

**用途**：记录每道往年考题的章节、知识点、题型映射。

**格式**：

```csv
年份,题号,所属章节,知识点,题型,使用公式或方法,是否高频,备注
```

**字段说明**：
- `年份`：试卷年份，如 2021
- `题号`：试卷中的题号，如 "一.1" 或 "3"
- `所属章节`：教材章节号，如 "第5章"
- `知识点`：该题考查的知识点名称
- `题型`：题型类别，如 "求极限"、"判断可微性"、"求切平面"
- `使用公式或方法`：解题使用的核心公式或方法名
- `是否高频`：是/否（出现在 ≥2 年试卷中即为高频）
- `备注`：补充说明

## 2. 覆盖审计文件

**路径**：`output/audit/chXX_coverage.csv`

**用途**：记录每章每个小节的处理决策及依据。

**格式**：

```csv
章节,教材页码,教材小节,处理方式,处理理由,证据,风险
```

**字段说明**：
- `处理方式` 仅允许三个值：
  - `保留`——完整写入教程
  - `压缩保留`——压缩为一两段概述
  - `剔除`——不写入教程正文
- `处理理由`：为什么这样处理（一句话）
- `证据`：必须写明依据来源，例如：
  - "用户明确要求剔除"
  - "往年卷未出现（2012-2021 共 10 年均未涉及）"
  - "课后习题未覆盖"
  - "高频出现在 2021 年第 3 题、2022 年第 2 题"
  - "是 XX 高频题型的前置知识"
  - "教材例题中反复使用"
- `风险`：`低` / `中` / `高`。如果某内容被剔除但存在误删可能，标为"中"或"高"

## 3. Markdown 自检文件

**路径**：`output/audit/chXX_md_check.txt`

**用途**：记录对教程 Markdown 文件的格式自检结果。

**格式**：

```txt
章节：第 X 章
检查时间：

一、标题层级检查
- 是否有一级标题（#）：
- 二级标题（##）数量：
- 三级标题（###）数量：
- 标题是否逐级嵌套无跳级：

二、数学公式检查
- $$ 是否成对出现：
- $$ 是否单独占行：
- 独立公式前后是否有空行：
- 公式块内部是否误含 $：
- 是否有 \(...\) 或 \[...\] 格式（禁止）：
- 中文是否误入公式环境：

三、图片链接检查
- 图片引用数量：
- 所有图片路径是否存在：
- 相对路径是否正确：
- 缺失图片列表：

四、结构完整性检查
- 是否包含"0. 本章考试相关性总结"：
- 是否包含"1. 本章知识结构"：
- 是否包含"2. 核心知识点"：
- 是否包含"3. 典型题型总结"：
- 是否包含"4. 本章复习清单"：

五、其他问题
- 列表/表格/代码块是否闭合：
- 是否有明显重复内容：
- 是否有断裂（引用但未定义的内容）：

六、结论
- 是否通过格式检查：
- 需要修复的问题：
```

## 4. 文本质量自检文件

**路径**：`output/audit/chXX_text_quality_check.txt`

**格式**：

```txt
章节：
检查时间：

一、本章常见考法覆盖检查
- 常见考法数量：
- 典型题型总结覆盖数量：
- 是否达到 80% 覆盖要求：
- 未覆盖题型：

二、必须掌握考点证据检查
- 每个必须掌握考点是否有往年卷、课后习题、教材例题或用户要求作为证据：
- 缺失证据的考点：

三、核心知识点解释检查
- 是否解释概念本质：
- 是否解释为什么引入：
- 是否解释公式每一项含义：
- 是否说明做题识别信号：
- 是否说明使用条件：
- 是否说明常见误区：
- 是否配有例题或反例：

四、例题质量检查
- 是否注明来源：
- 是否注明对应知识点：
- 是否解释第一步为什么这样做：
- 是否解释中间步骤为什么成立：
- 是否说明答案如何检查：
- 是否说明常见错误：

五、结论
- 是否通过文本质量检查：
- 需要修复的问题：
```

## 5. 图片审计文件

**路径**：`output/audit/chXX_image_audit.csv`（仅当本章使用图片时生成）

**格式**：

```csv
章节,图片文件名,教材页码,图号或表号,对应知识点,处理方式,是否局部裁剪,是否含无关正文,是否清晰,是否通过检查,备注
```

**字段说明**：
- `是否局部裁剪`：是 / 否
- `是否含无关正文`：是 / 否
- `是否清晰`：是 / 否
- `是否通过检查`：是 / 否
- 如果 `是否局部裁剪=否` 或 `是否含无关正文=是`，则 `是否通过检查` 必须为"否"
